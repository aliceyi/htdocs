旋转组件

JS创建方法

$('#divChartShares').visualize({
            type : "pie",
            height : 250,
            width:500,
            chartId: "PieCon",
            appendTitle:false,
            appendKey:true
        });

属性

type:string	设置图表类型，'bar', 'area', 'pie', 'line' 默认: 'bar'.

width: number 设置图表的宽度，默认是table的宽度

height: number 设置图表的高度，默认是table的高度

appendTitle: boolean 设置图表是否显示title, 默认是false

title: string 设置图表的标题，默认是table的Caption元素

colors: array 数组项的值是十六进制，设置图表的外观，默认['#be1e2d','#666699','#92d5ea','#ee8310','#8d10ee','#5a3b16','#26a4ed','#f45a90','#e9e744']

textColors: array  数组项的值是十六进制, 图表的每项文字颜色对应数组的颜色，如果对应项为null/undefined 则用样式的默认颜色，默认是[]

parseDirection:string 设置table里的数据显示的方向，如 'x' and 'y'，默认是 'x'

pieMargin: number 设置饼图的周围空间，默认是 20

pieLabelPos: string  设置文字在饼图的位置，'inside' and 'outside' 默认是'inside'

lineWeight: number 设置图表线的粗细在线图和面积图中，默认是'4px'

barGroupMargin: number 设置图表每个柱子之间的间距，默认是 10

barMargin: number. Creates space around bars in bar chart (added to both sides of each bar). Default: 1

注意点

若图表支持IE需引入excanvas.js
