分页组件

JS创建方法

var opt = {
                form:$("#formId"),
                pageDesCon:$("#pageDesCon"),
                next_text: "下一页",
                current_page: 1,
                items_per_page: 10,
                num_display_entries: 5,
                num_edge_entries: 2,
                prev_text: "上一页"
            };
$("#pagination").pagination(100, opt);

属性

pageDesCon:Object 设置显示分页信息的容器

next_text:String 设置下一页按钮显示的文字

prev_text:String 设置上一页按钮显示的文字

current_page:Integer 设置分页从第几页开始显示

items_per_page:Integer 设置分页显示数据的条数

num_display_entries:Integer 设置分页按钮显示的页数，其余用“...”代替

num_edge_entries:Integer 设置分页按钮最后显示的页数

方法

pagination(int,object): 创建分页的方法(总分页数量，设置分页组件的参数对象)
